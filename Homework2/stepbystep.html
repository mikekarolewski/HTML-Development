Step 1:

First, from the "File" menu, select "New", this will display many different options. The one you want to select will be "Project". This will be shown in the following screenshot.

Step 2:

After selecting "Project", you will be faced with many different folders. The one you will be most concerned with is "Web". Click "Web", then select "Dynamic Web Project", and finally the "Next>" button, which is shown here.

Step 3:

Now that your project is created and should show up in your workbench, the next step is to create the HTML file. To do this, right click on the project, and select "HTML File", as shown.

Step 4:

We've already created an HTML file, so why not add some flavor to it? With CSS styling of course! CSS is a great way to style your web page and you can easily create a CSS type file by right clicking the project, hitting "File" and when you save the file, save it under the suffix ".css". For example, a CSS type file would look like "style.css".
